Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Declarations are not case sensitive.
  • Section elements are semantic elements that define a section in a document that contains thematic content.

CSS

  • Used for the presentation aspects of web applications.
  • One of three cornerstone languages used in web development.

Git

  • Version control system that lets you manage and keep track of each version of your project.
  • Github is the web interface for Git.
  • Allows multiple people to make seperate changes to a program at the same time.
  • git status: checks what branch we are currently on
  • it checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • The 7 different types in JavaScript are string, number, object, boolean, null, undefined and symbol.
